# POLS 6382: Quantitative Methods III

This repository contains materials for **TA-led lab sessions and homework support** for  
**POLS 6382 – Quantitative Methods III (Maximum Likelihood Estimation)**  
at the **University of Houston**, **Fall 2025**.

**Instructor:** Ling Zhu  
**Teaching Assistant:** Songeun Emily Lee


## Contents
- Lab session materials
- Homework assignments
- Annotated example scripts (R)
- Supplementary code and demonstrations used in lab

> Some scripts reflect *instructional examples* discussed in lab sessions.  
> They are provided for learning and reference purposes only.

## Topics Covered
- Maximum Likelihood Estimation (MLE)
- Binary choice models (logit, probit)
- Ordered and multinomial response models
- Count data models (Poisson, Negative Binomial)
- Sample selection models
- Event history / survival analysis

## Software
All materials are implemented in **R**.  
Students are expected to be familiar with basic R usage and R Markdown / LaTeX.

## Intended Use
This repository is designed for **educational and instructional purposes**.  
Students should develop and submit their **own original solutions** for coursework.

Unauthorized use, reproduction, or submission of these materials as part of
graded coursework—whether in this course or others—is not permitted.
